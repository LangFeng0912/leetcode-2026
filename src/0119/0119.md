- 解法：
    - java 位运算符: `a | b`
    - `ans+1` 将ans中从低位到高位第一个 0 变成 1 并且使该 0 之前的全部低位 1 变为 0， 
    - `ans∣(ans+1)`: 将ans 中从低位到高位第一个 0 变成 1
    - 找出 x 中第一个0的位置pos， 将`pos−1`处的 1 变为 0
        - 寻找第一个0   
            - 如果 x & 2^K (只有最高位是1) == 0 则说明该位是0， 直接结束了，跳出循环取res
            - 预设前一步存的res
            - 否则 继续循环， d从1开始 不断的左移 d<<=1
