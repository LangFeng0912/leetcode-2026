- 前缀和：一个数组 两个坐标i和j之间所有元素的和
    - 衍生出二维矩阵的前缀和：比如求一个矩形(正方形)的所有元素之和
    - 每个位置的前缀和
    ```
    prefix[i][j] = 
        curVal[i][j] 
        + prefix[i-1][j] 
        + prefix[i][j-1] 
        - prefix[i-1][j-1]
    ```
    - 由此得出这个矩形的面积 比如从(x1,y1)到(x2,y2)， 从而计算出每个矩形的面积，继续运算
    ```
    size = prefix[x2][y2] 
        - prefix[x1-1][y2]
        - prefix[x2][y1-1]
        + prefix[x1-1][y1-1]
    ```
