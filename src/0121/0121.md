- [347. 前 K 个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/description/)
    - 大顶堆 小顶堆： 只维护前K个元素
    - java 实现 `PriorityQueue<> minHeap`
        - 入堆: `heap.add()`
        - 堆顶: `heap.peek()`
        - 删除堆顶: `heap.poll()`
    - 思路： 堆的size小于k时，直接入堆；大于K时，堆顶出堆（小顶堆意味着最小的元素出堆），当前元素入堆

- 注意哈希表，`map.keySet`, `map.put()`,
