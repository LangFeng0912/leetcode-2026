- [98. 可达路径](https://kamacoder.com/problempage.php?pid=1170)
- [ ] [https://leetcode.cn/problems/all-paths-from-source-to-target/description/]

- 图论 + 深度搜索
    - 首先建立图的邻接矩阵
    - 全局变量，path和result
    - dfs(图， 节点)
        - 已经访问过了：return
        - path 加入当前节点
        - 确定终止条件
            **注意result数组加入时 不能直接加path， 而是一个副本**
            - 打印/加入result； return
        - 遍历当前节点的所有可能下一步的下一个节点
            - 操作节点
            - dfs(图，节点)
            - 回溯（从path矩阵移除）

- 岛屿问题：
    - 终止条件：
        - 边界(0<=i,j<=m&n>)
        - 遇到河流： 0
        - 遇到visited
    - visited元素 变为true
    - 上 下 左 右

- 最大岛屿 dfs 返回 int
- 岛屿数量 在外层for循环里 统计count

- 如何存取邻接矩阵: `List<Integer, List<Integer>> adj`
- visited 是一个数组 而非hashmap
- 如何判断 是都有回溯操作：看是否需要记录每一条路径
